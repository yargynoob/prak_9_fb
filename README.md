### Настройка проекта
1. Клонируем проект
   ```shell
   git clone ttps://github.com/yargynoob/prak_9_fb.git
    ```
3. Запустить **Docker**
4. Прописать команду
    ```shell
    docker-compose up --build 
    ```
5. На `http://localhost:3030` находится страницу админа
6. На `http://localhost:8100` находится страница пользователя
